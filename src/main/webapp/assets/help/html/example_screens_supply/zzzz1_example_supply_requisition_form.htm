Example Requisition Form

Below is an example Requisition Form screen which is used by authorized office employees only. The Requisition Form is used to order office products that are kept in stock by the Senate Supply office.

Fields Explained

Refer to the chart to find basic definitions for fields.

Field Name

Definition

Destination [change]

This is the office location for whom the order was placed and where it will be delivered. Click the [change] link to change the location code displayed.

Search

This field is used to search for Senate items in stock.

Sort By

This drop-down field enablse you to sort supply items by name or category.
